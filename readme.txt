Publishing statistics for each blog in your WordPress Multisite Network

== Description ==
You'll get aggregated posts publishing statistics for each site in your WordPress Multisite Network on the single screen.

After activation navigate to Network King to see your detailed Network publishing statistics.

== Installation ==
1. Login to your WordPress Multisite admin area
2. Search for Network King
3. Click install Network King
4. Activate through the Plugins menu in WordPress or when asked during installation
5. See your Network Publishing Stats from the Network King admin area

Alternatively:
1. Upload plugin zip to /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress Network Admin
3. See your Network Publishing Stats from the Network King admin area
